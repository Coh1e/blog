# Colophon

## 字体

- **正文（拉丁）**：[Newsreader](https://fonts.google.com/specimen/Newsreader)，Production Type 设计，Google Fonts 免费
- **正文（中文）**：[LXGW WenKai 霞鹭文楷](https://github.com/lxgw/LxgwWenKai)，开源
- **标题 / UI**：[Inter](https://rsms.me/inter/)，开源
- **代码**：[JetBrains Mono](https://www.jetbrains.com/lp/mono/)，开源

- **生成**：Vite + React + react-router + vite-react-ssg

- **搜索**：Pagefind
- **数学公式**：KaTeX
- **图表**：Mermaid（运行时渲染）
- **Hosting**：Cloudflare Pages
- **DNS**：Cloudflare
