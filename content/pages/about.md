# 关于这个站

未经授权 · Unauthorized 是一个独立的写作站点，记录长一点的想法（[[posts/markdown-blog|文章]]）和短一点的笔记（[[notes/digital-garden|笔记]]）。

页面用 React + Vite 静态生成，源代码不依赖任何后台服务。中英双语并行：中文在 `/`，英文在 `/en/`。

设计灵感：Paul Graham 的留白、Gwern 的密度、Andy Matuschak 的笔记园艺、谢益辉的安静学者气。

如果想联系我，最好的方式是 GitHub 上发 issue。
