I've stopped looking for the "best" tool to ship a small thing. The fastest path, every time, has turned out to be the boring tool I already know.

A static site I could build in one evening with Quartz vs. a static site I could "really nail" with a custom Astro setup. Quartz wins, even though Astro is technically more flexible. The flexibility is dead weight when the actual goal is to ship a small thing this weekend.

This is one of those notes that looks obvious in retrospect but kept tripping me up for years. The pull toward "the right tool" is strong because it _feels_ like the responsible choice. It is, for projects you'll be running for a decade. It isn't, for projects you want done by Sunday.
