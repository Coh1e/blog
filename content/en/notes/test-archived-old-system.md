> [!WARNING]
> This note is `archived`: the system it describes is no longer in use. I keep it so that backlinks from [[en/notes/note-taking-is-not-knowledge|the current note-taking note]] still have somewhere to land.

I used Obsidian + Templater + Dataview for nearly two years. I retired it for three reasons:

1. The system itself got **bloated** — 20+ plugins, 6-second startup, 30 minutes to set up on a fresh machine.
2. The data was Markdown but **bound to Obsidian's renderer** — `[[]]` wikilinks were dead in any other Markdown viewer.
3. I started spending more time **tuning the system** than writing notes. That's the killer signal.

The replacement: see [[en/notes/note-taking-is-not-knowledge|the latest version of note-taking]], stripped down to plain markdown files + git.
